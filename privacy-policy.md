### Privacy Policy

Muslim Prayer Reminder does not collect, store, or share any personal user data.

The extension retrieves prayer time information from a public prayer time API in order to calculate and display daily Salah (Namaz) reminders. This request does not include personally identifiable information.

Any settings or preferences chosen by the user (such as reminder preferences) are stored locally in the browser using Chrome's storage API and are not transmitted to any external servers.

The extension does not sell, transfer, or use user data for advertising or tracking purposes.
